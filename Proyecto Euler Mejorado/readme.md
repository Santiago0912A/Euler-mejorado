## Introduccion

El metodo numerico euler mejorado es uno de los mas exactos en cuanto 
a la aproximacion de la raiz de una ecuacion diferencial. Con este pr
ograma podras aproximar la raiz de tu ecuacion con los parametros del
problema que necesites. Esperemos lo disfrutes y lo utilices para el 
bien :), Saludos!!
